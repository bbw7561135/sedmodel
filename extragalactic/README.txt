#### How to execute the code #####
g++ ic_spectrum_new.C 
./a.out InputICSSC > data_file.txt
The oouput will be stored in data_file.txt
----------------------------------------
#### About data file #############
1st column: frequency (Hz)
2nd column: energy (eV)
3rd column: Synchroton SED (TeV/cm^2/s)
4th Coulmn: IC SED (TeV/cm^2/s)
5th column: IC with ebl absorption (TeV/cm^2/s)
6th column: Synchrotron + IC SED (TeV/cm^2/s)
7th column: Synchrotron + IC with ebl absorption SED (TeV/cm^2/s)
